Abstract This paper provides a precise result which shows that insensitivity to small measurement errors inclosed-loop stabilization can be attained provided that the feedback controller ignores observations

1 Introduction We deal in this paper with control systems

.x = f (x, u) (1) for which states x(t) evolve in X = Rn and controls take values u(t) in a metric space U (precise definitions are given later). We suppose that there is some compact set A to which we would like to asymptotically drive all states. A special case is that in which A = {0}. Our results are new for that particular case, but we state and prove them in more generality, since the proofs are not harder for a general compact invariant set (such as a periodic trajectory, see Figure 1).

In this paper, we will be interested in fact in a somewhat stronger notion of stabilization, in which measurement noise may occur. That is, the controller measures x(t) + e(t) at time t, where e(t) is an "observation error" and one wants that all trajectories converge to A provided that the errors e(t) be sufficiently small. The precise value of e(t) is unknown by the controller, but one uses information about upper bounds on the magnitude of e(t) when carrying out the design. The meaning of "small" will be discussed later, but it is clear that only small errors can be tolerated when close to A.

Of course, when the feedback law k is a continuous function, this insensitivity, or robustness, to small errors is automatic. To see this, we may consider the perturbed system

.x = f (x, k(x + e)) (3) which arises when the control u(t) = k(x(t) + e(t)) is used instead of u(t) = k(x(t)). If k was a stabilizing continuous feedback, then, since f (x, k(x + e)) ss f (x, k(x)) when e is small, the movement of the trajectories is still in the correct direction. This can be proved rigorously in several manners, for instance using Lyapunov functions. Indeed, suppose that V is a Lyapunov function for verifying the asymptotic stability of A with respect to the closed-loop system .x = f (x, k(x)) (see e.g. [7]); thus,r

But this proof breaks down when k is not continuous, because the vector f (x, k(x + e)) may point in a very different direction than f (x, k(x)).

Discontinuous feedbacks and chattering Unfortunately, discontinuous feedback laws k are often unavoidable. For example, for the system .x = u with X = U = R, the set A = {-1, 1} can be made globally asymptotically stable under many different discontinuous feedback laws (such as: k(x) = 1 - x for x >= 0 and k(x) = -1 - x for x < 0) but there is no possible continuous global feedback stabilizer k(x) (because the domain of attraction of A has two connected components).

A hybrid strategy There are many techniques that can be used in order to avoiding chattering. One such technique, which can be applied in great generality, is based upon the following modification of a given control strategy k which might not be robust to measurement errors. Starting at some initial time, let us say t0 = 0, we measure k(x(0) + e(0)), obtaining a control value u0. We now apply the constant control u(t) j u0 during a small interval [0, t1). This procedure is then repeated: at time t1 we resample, obtaining some u1 := k(x(t1) + e(t1)), and apply this control during another small interval [t1, t2), and so on. This modified strategy is not a pure "continuous time" one, in that a minimum intersample time is required. It can be interpreted, rather, as constructing a hybrid (different time scales needed) and dynamic (requiring memory) controller.

Using still the simple example of .` = u in the circle as an illustration, this modified strategy would mean that we decide in which direction to move (clockwise or counterclockwise), based on the current measurement (which may well be somewhat inaccurate). Now, we continue moving in the chosen direction, not paying any attention to observations, for a small time interval. Only after that interval do we again sample. Suppose that our original observation k(x(0) + e(0)) tells us that we appear to be near the top point, p0. Depending on an assumed upper bound on the magnitude of the measurement errors, we can estimate an interval around p0 where the true state lives. Now we move away and towards the lower point for a certain minimal time whose length depends on this interval. We end up sufficiently far from the discontinuity. At this point, we know for sure on which side of the top we are. Actually, it really doesn't matter if we moved clockwise or counterclockwise; in either case, we will be far enough from p0 to be sure that errors in the next measurement k(x(t1) + e(t1)) do not lead to catastrophic consequences.

This example is too simple to exhibit the complexity of the general result, especially because there is only one discontinuity, at p0, in the feedback law, and we are safe once that we are far from p0 (and, since motions directed by k are in the direction away from p0, we never go back towards it, if we started far enough). However, it illustrates the general principle that the length of time that we wait must be in some sense proportional to the uncertainty. Actually, the general case is not too difficult to understand. We now explain it intuitively.

The role of Lyapunov functions We discussed already the fact that the stability of the unperturbed system (2) implies that of (3) when k is continuous. As a matter of fact, the same implication is true in the more general situation in which there is some continuously differentiable Lyapunov function for the closed-loop system. Basically, this follows from the approximation

rV (x) * f (x, k(x + e)) ss rV (x + e) * f (x + e, k(x + e)) < 0 , using the Lyapunov property at the point x + e instead of at x, and not using continuity of k. The proof is given in [6] -- see also [11] -- where it is also shown that, in general, differentiable Lyapunov functions do not exist. Observe that for our circle example, this alternative argument cannot be applied; the obvious "Lyapunov function" given by the geodesic distance to p measured along the circle is not a differentiable function, since it is nondifferentiable at the conjugate point p0. However, continuous (but nondifferentiable) Lyapunov functions do play a role.

4 Non-Global Stability and Systems on Manifolds A local version of Theorem 1 holds. Suppose that (1) is locally asymptotically controllable to the compact and zero-invariant set A, meaning that there are a compact subset U0 ` U, a constant ffi > 0, and a fi 2 KL, so that for each x0 with |x0| < ffi there is some U0-valued control u so that x(t, x0, u) exists for all t >= 0 and satisfies (4). Then, there are ffi0 > 0, \Gamma  2 K1, and for each pair " < K with " > 0 and K < ffi0 there are a feedback law k, and ffi, ^, T , such that, whenever d(ss) <= ffi, |e(t)| <= ^d(ss) for all t >= 0, and |x0|A <= K, the solution xss(t, x0, e) is well-defined and satisfies |x(t)|A 2 \Gamma (K) for t >= 0 and |x(t)|A <= " for t >= T . This is a consequence of the global result, as we sketch next.

We first introduce a new system which is globally asymptotically controllable to A and which reduces to the original system in the neighborhood BAffi/2. Such a system can be easily obtained in

the form .x = f (x, u) + '(x)v, where ' is a smooth function which equals 1 outside BA2ffi/3 and is zero on BAffi/2, and v 2 Rn represents a set of n additional controls. Now we apply the global Theorem to this extended system. Finally, we pick ffi0 small enough so that \Gamma (ffi0) < ffi/2, which insures that when starting in a state with |x0|A <= ffi0 the trajectories will remain in BAffi/2 and hence be trajectories of the original system.

More generally, one may give a version of our main result for systems in which the state space is a general (second countable) differentiable manifold X instead of Rn (one restates the stability definitions in topological terms, independent of a metric). Of course, the meaning of "measurement error" must be clarified, since the sum x(t) + e(t) makes no sense if X is not a vector space. It should be interpreted as using feedback u = k(z(t)), where z(t) is close to x(t) in a suitable metric. We sketch next a possible way to achieve such a generalization.

Given a system on X, which is asymptotically controllable to the compact subset A, we embed X in an Euclidean space Rk, and extend the dynamics to Rk in such a manner that (1) when restricted to X we recover the original dynamics, and (2) the extended system is still asymptotically controllable to A. Now, a feedback law for the extension restricts to one for the original system, and it has the same robustness properties as the extension.

The construction of this extension can proceed as follows. We first embed X as a closed submanifold of Rk (Whitney), and extend the original system to a tubular neighborhood of X. On this tubular neighborhood, we modify the dynamics by adding a term that attracts (contracts) to X in the normal direction (and so is zero on X). This gives an extension to the tube, preserving controllability. Finally, on a closed set disjoint from X but suitably overlapping the tube, we add a term (as "+'(x)v" above) which allows one to control all states in the embedding space Rk into the tube. We leave the details for future work.
